## 시작 가이드
### Requirements
- Django 3.2.19
- mysqlclient 2.2.1
- Pillow 10.1.0
- django-axes 6.3.0


### Installation
```shell
$ git clone https://github.com/ShieldLetter/ShieldLetter.git
$ cd ShieldLetter
```

### MySQL Server
```sql
-- 스키마 생성
create schema shieldletterdb default character set utf8;

-- 스키마 사용
use shieldletterdb;

-- 유저 생성
CREATE USER 'user'@'localhost' IDENTIFIED BY 'password';

-- 유저에게 권한 부여
GRANT ALL PRIVILEGES ON shieldletterdb.* TO 'user'@'localhost';
FLUSH PRIVILEGES;
```

### settings.py 설정
`ShieldLetter/config/settings.py` 파일에 `DATABASES`의 `local env` 부분의 주석을 해제하면 됩니다.

```python
# local env
DATABASES = {
     'default': {
         'ENGINE': 'django.db.backends.mysql',
         'NAME': 'shieldletterdb',
         'USER' : 'user',
         'PASSWORD' : 'password',
         'HOST' : '127.0.0.1',
         'PORT' : '3306',
         'OPTIONS': {
             'init_command': "SET sql_mode='STRICT_TRANS_TABLES'",
         },
     }
}
```

### App Run
```python
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver
```

### Django 관리자 계정 생성 방법
```python
$ python manage.py createsuperuser
```
